# 要求:

背包里拥有道具 - 旅行者之书: 村庄移动 [81186, 60489]

# 命令与功能

- bs 进入对应地图后使用

- bsto [数字] 在不同的地图,有不一样数量的点位. 例如 传送到3号boss刷新点: bsto 3

- tpto [数字] 开启 村庄移动 后执行对应[村庄]编号传送. 例如 传送到希望村庄: tpto 79001
